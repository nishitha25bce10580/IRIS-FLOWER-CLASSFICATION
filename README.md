# IRIS-FLOWER-CLASSFICATION
This project uses Machine Learning to classify iris flowers based on sepal and petal measurements. Data analysis and visualization were done using Pandas, Seaborn, and Matplotlib, and a Decision Tree model was trained using Scikit-learn. The model accurately predicts the flower species from input data.

## 📌 Overview

This project is a Machine Learning model that classifies iris flowers into three species:

* Setosa
* Versicolor
* Virginica

The classification is based on four input features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

## 🛠️ Technologies Used

* Python
* Pandas

* Seaborn
* Matplotlib
* Scikit-learn

---

## 📊 Dataset

The dataset was obtained from an external CSV file (Iris dataset).
It contains 150 samples with 4 features and 3 target classes.

---

## ⚙️ Workflow

1. Data Loading using Pandas
2. Data Exploration and Cleaning
3. Data Visualization using Seaborn and Matplotlib
4. Feature Selection
5. Train-Test Split
6. Model Training (Decision Tree Classifier)
7. Model Evaluation (Accuracy Score, Confusion Matrix)
8. Prediction on new input data

---

## 📈 Results

* Model achieved high accuracy (~90–100%)
* Successfully predicts flower species based on input features
